# Описание проекта BIOMAX AI

**BIOMAX AI** — мультиагентная платформа «персональной операционной системы здоровья»: 15 специализированных AI-агентов анализируют сон, питание, тренировки, анализы крови, эпигенетику и другие данные из 100+ интеграций (wearables, лаборатории, CGM, геномика), дают персонализированные рекомендации и инсайты, а Custom RAG позволяет подключать собственную базу знаний (протоколы, исследования). Платформа ориентирована на российский рынок с соблюдением 152-ФЗ, поддержкой GigaChat/YandexGPT и интеграциями с локальными лабораториями; MVP предусматривает запуск через Telegram Bot, затем WebApp и мобильное приложение на Flutter.
