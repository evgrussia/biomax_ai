# Промпты для Figma Make — BIOMAX AI Personal Dashboard (Demo)

**Проект:** BIOMAX AI v2.0 — Personal Health Operating System  
**Дата:** 2026-01-28  
**Цель:** Демо-версия личного кабинета с онбордингом и моковыми данными  
**Режим:** Демо (без авторизации, вход с промо-сайта)  
**Старт проекта:** Апрель 2026

---

## КОНЦЕПЦИЯ ДЕМО ЛИЧНОГО КАБИНЕТА

**Цель демо:** Дать пользователю "пощупать" интерфейс системы, ввести свои первичные данные для онбординга, увидеть как будет выглядеть персонализированный dashboard — но с моковыми данными и сообщением о демо-режиме.

**Flow:**
1. Кнопка "Попробовать демо" на промо-сайте
2. Welcome-экран демо личного кабинета
3. Онбординг (4-5 шагов ввода данных)
4. Dashboard с моковыми данными + баннер "Демо"
5. Интерактивное исследование функционала
6. Modal: "Полный функционал доступен после запуска (апрель 2026)"

---

## ГЛОБАЛЬНЫЕ СТИЛЕВЫЕ УКАЗАНИЯ (DASHBOARD)

Используй для ВСЕХ экранов личного кабинета:

```
Стиль: Futuristic AI Health Dashboard 2026
- Dark mode primary (#0A0F1E → #1A1F2E gradient)
- Акценты: Electric cyan (#00D4FF), Bioluminescent green (#00FF94), Soft purple (#8B5CF6)
- Glassmorphism: backdrop-blur(20px), полупрозрачные карточки с border glow rgba(255,255,255,0.1)
- Sidebar navigation: темная колонка слева (240px ширина)
- Content area: светлее на 1-2 тона, с карточками метрик

- Cards: border-radius 16px, subtle shadow, hover states
- Демо-баннер: жёлтый/оранжевый акцент, чтобы выделяться
```
