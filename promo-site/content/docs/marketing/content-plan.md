## Content Production Workflow

### Blog Post Workflow

```
1. Topic Research (1 day)
   - Keyword analysis
   - Competitor content audit
   - Outline creation
   
2. First Draft (2-3 days)
   - Writer creates draft
   - Include research citations
   - Add CTAs
   
3. Review & Edit (1-2 days)
   - Content lead review
   - Expert review (if technical)
   - SEO optimization
   
4. Design Assets (1 day)
   - Featured image
   - Infographics
   - Screenshots
   
5. Final Review (1 day)
   - Grammar/spelling
   - Fact-check
   - Link check
   
6. Publish & Promote (1 day)
   - Schedule publication
   - Social promotion
   - Email (if applicable)

Total: 7-10 days per post
```

### Video Production Workflow

```
1. Planning (2 days)
   - Script writing
   - Storyboard
   - Resource gathering
   
2. Production (1-2 days)
   - Recording
   - Screen capture
   - B-roll
   
3. Post-Production (3-5 days)
   - Editing
   - Graphics/animations
   - Sound design
   
4. Review (1 day)
   - Feedback
   - Revisions
   
5. Publish (1 day)
   - YouTube upload
   - Thumbnail
   - Description & tags

Total: 8-11 days per video
```

## Content Quality Guidelines

### Writing Standards

1. **Accuracy** — все утверждения подкреплены источниками
2. **Clarity** — простой язык, избегаем жаргона (или объясняем)
3. **Structure** — чёткая иерархия заголовков, короткие параграфы
4. **Value** — каждый пост отвечает на конкретный вопрос читателя
5. **CTA** — каждый пост имеет призыв к действию

### Scientific Content Standards

1. **Source attribution** — ссылки на PubMed, DOI
2. **Evidence levels** — указывать тип исследования (RCT, observational)
3. **Disclaimers** — образовательный характер, не замена врачу
4. **Balanced view** — представлять разные точки зрения
5. **Updates** — актуализировать при новых исследованиях

### Brand Voice

| Attribute | Description | Example |
|-----------|-------------|---------|
| **Экспертный** | Глубокое понимание темы | "Исследование показало..." |
| **Доступный** | Объясняем сложное просто | "Проще говоря..." |
| **Практичный** | Конкретные рекомендации | "Попробуйте сегодня..." |
| **Честный** | Не обещаем невозможного | "Результаты могут отличаться..." |
| **Современный** | В курсе последних трендов | "Новое исследование 2026 года..." |
