# Commands (конвенции чата)

Cursor не требует отдельной регистрации команд: мы используем **префиксы в сообщениях** (см. `.cursor/rules/00-agentic-system-core.md`).

## Список

- `/start-project <идея>`
- `/status`
- `/route <agent> <task>`
- `/checkpoint`
- `/summary`

Каждый файл в этой папке — подсказка по входу/выходу команды и ожидаемым артефактам.
