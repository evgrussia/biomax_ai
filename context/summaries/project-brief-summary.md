# BIOMAX AI v2.0 — Summary

**Status:** Discovery | **Updated:** 2026-01-26

## One-Liner
Personal Health Operating System — мультиагентная AI-платформа для биохакинга и управления здоровьем.

## Core Innovation
- **15 AI-агентов** (Nutrition, Sleep, Cognitive, Fitness, Longevity, Mental Health, etc.)
- **100+ интеграций** (wearables, CGM, labs, genomics, microbiome)
- **Custom RAG** — пользователи загружают свои протоколы и исследования
- **8 измерений wellness** → Health Score 0-100
- **Safety-first** — медицинские границы, контрпоказания, red flags

## Target Audience
1. Продвинутые биохакеры (множество устройств, протоколов)
2. Health optimizers (активно работают над здоровьем)
3. Longevity-focused (биологический возраст, эпигенетика)

## Business Model (Freemium SaaS)
| Tier | Price | Target |
|------|-------|--------|
| Free | 0₽ | Lead gen, 100K users |
| Optimize | 990₽/мес | Health optimizers, 30K |
| Biohacker Pro | 2,490₽/мес | Advanced biohackers, 15K |
| Longevity Elite | 9,990₽/мес | Premium longevity, 3K |
| Enterprise | Custom | B2B corporate wellness |

**Additional Revenue:** Marketplace (10-15%), Lab referrals (5-8%), Premium Protocols

**Target M24:** 60M ₽ MRR, 40K paying users, LTV/CAC 7x

## Tech Stack
LangGraph + FastAPI + PostgreSQL + ClickHouse + Qdrant + Neo4j + Flutter + Next.js

## Key Risks
- Регуляторные ограничения (wellness ≠ medicine)
- Data security (152-ФЗ, персональные медданные)
- AI-галлюцинации в медицинском контексте
- Низкая конверсия Free→Paid
- Конкуренция BigTech

## Key Partnerships (Target)
- Labs: INVITRO, Helix, CMD
- Wearables: Oura, Garmin, WHOOP
- LLM: GigaChat, YandexGPT
- Genomics: 23andMe, Atlas Biomed
