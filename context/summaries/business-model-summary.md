# Business Model Summary: BIOMAX AI v2.0

**Status:** Draft | **Updated:** 2026-01-26

## Business Model Type
**Freemium SaaS** с градацией тарифов + дополнительные revenue streams

## Revenue Streams (Target M24: 60M ₽ MRR)

| Stream | % Revenue | Model |
|--------|-----------|-------|
| Subscriptions B2C | 70% | Recurring |
| Enterprise B2B | 15% | Recurring + Services |
| Marketplace | 8% | Commission 10-15% |
| Lab Referrals | 4% | Commission 5-8% |
| Premium Protocols | 3% | One-time |

|--------|-------|

## Customer Segments

1. **Продвинутые биохакеры** (50% revenue) — 5+ устройств, N=1 эксперименты
2. **Health Optimizers** (30%) — Apple Watch, ищут AI-коуча
3. **Longevity-focused** (10%) — биовозраст, эпигенетика
4. **B2B Corporate** (10%) — HR wellness programs

## Value Proposition

| Segment | Core Value |
|---------|------------|
| Biohackers | Custom RAG + все данные в одном месте |
| Optimizers | AI-коуч с персонализацией |
| Longevity | Digital Twin + эпигенетика |
| B2B | ROI wellness + analytics |

## Key Partnerships

- **Labs:** INVITRO, Helix, CMD
- **Wearables:** Oura, Garmin, WHOOP
- **LLM:** GigaChat, YandexGPT
- **Cloud:** Yandex Cloud (152-ФЗ)

## Cost Structure (M24: 23M₽/мес)

| Category | % |
|----------|---|
| Team | 60% |
| LLM APIs | 15% |
| Infrastructure | 12% |
| Marketing/CAC | 8% |
| Other | 5% |

## Competitive Advantage

1. **Full biohacking coverage** (vs point solutions)
2. **15 AI agents** (vs rule-based)
3. **Custom RAG** (unique — no competitors)
4. **RU localization** (152-ФЗ ready)

## Key Risks

- Регуляторные ограничения → wellness positioning
- Низкая конверсия → value demonstration
- AI галлюцинации → RAG + Safety Agent
